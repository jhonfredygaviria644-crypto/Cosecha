Sincronizar entre tablets (Google Sheets)

Conecta esta calculadora a una Hoja de Google compartida para que los muestreos ingresados desde cualquier tablet se vean en todas las demás.

1. Crea el puente con Google Sheets (una sola vez)

Hazlo una sola vez, desde una computadora. Después, todas las tablets solo necesitan pegar la URL que te da el paso 4.

  1. Crea una Hoja de Google nueva y vacía (sheets.google.com).
  2. Ve a Extensiones → Apps Script. Borra el código de ejemplo y pega el script que te descargué abajo.
  3. Guarda, y presiona Implementar → Nueva implementación → tipo "Aplicación web". Configura: Ejecutar como "Yo", Quién tiene acceso "Cualquier usuario".
  4. Copia la URL de la aplicación web que te entrega Google y pégala abajo, en cualquier tablet.

2. Conecta esta tablet

Sin conectar todavía — los muestreos solo se guardan en este dispositivo.

3. Sincronizar

Al guardar un muestreo, esta tablet intenta subirlo automáticamente. Usa estos botones para forzar la sincronización completa.

4. Cómo revisar que ya quedó subido

• ☁ Subido: ya está confirmado en la Hoja de Google — cualquier otra tablet puede verlo.
• ⏳ Pendiente: se guardó en este dispositivo pero aún no se confirmó en la hoja (revisa tu internet y presiona "Subir mis muestreos guardados aquí").
• También puedes abrir directamente la Hoja de Google desde cualquier computador y ver las filas que van llegando.
